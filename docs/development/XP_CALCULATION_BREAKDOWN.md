## Savage Worlds XP Cost Rules

### Attributes
- **Cost:** 2 XP per die step above d4
- **Die Progression:** d4 → d6 → d8 → d10 → d12
- **Examples:**
  - d4 = 0 XP (starting value)
  - d6 = 2 XP (1 step × 2)
  - d8 = 4 XP (2 steps × 2)
  - d10 = 6 XP (3 steps × 2)
  - d12 = 8 XP (4 steps × 2)

### Skills
- **Below linked attribute:** 1 XP per die step above d4
- **At or above linked attribute:** 2 XP per die step above d4
- **New skill starts at d4:** First raise to d6 costs 1 XP if below attribute

### Edges
- **Cost:** 2 XP each (flat rate)

### Starting Characters
- Characters begin with:
  - 5 attribute points (can raise attributes from d4 baseline)
  - 15 skill points (each point = one die step)
  - Hindrances give bonus points (Major = 2 points, Minor = 1 point)

**Note:** The XP calculations below represent the cost to BUILD these characters from a baseline d4/d4/d4/d4/d4 character with no skills or edges. This gives us a baseline for tracking future advancement.
