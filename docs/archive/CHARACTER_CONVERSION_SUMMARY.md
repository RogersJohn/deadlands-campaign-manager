## Conversion Rules

### Attribute Mapping
```
Deadlands Classic (8 attributes) → Savage Worlds (5 attributes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Cognition     ┐
Smarts        ├→ Smarts (use higher value)
              ┘
Deftness      ┐
Nimbleness    ├→ Agility (use higher value)
              ┘
Quickness     → (dropped - was reflexes/initiative)

Spirit        → Spirit (unchanged)
Strength      → Strength (unchanged)
Vigor         → Vigor (unchanged)
```

### Die Notation Conversion
```
Deadlands Classic → Savage Worlds
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1d4, 1d6            → d4   (minimal)
1d8, 1d10, 2d6      → d6   (trained)
1d12, 2d8, 2d10, 3d6 → d8  (good)
2d12, 3d8, 3d10, 4d6 → d10 (very good)
3d12+, 4d10+, 5d6+   → d12 (expert)
```

3. ✅ Convert all skill die values to Savage Worlds format
